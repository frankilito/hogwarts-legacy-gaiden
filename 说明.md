# 霍格沃茨之遗 · 番外篇(1927)

> 一款魔法学院生活 RPG 同人作品 —— 不是魔法学校的壳子,而是一段有日程、有朋友、有秘密的学院生活。
> 非商业粉丝创作 · 使用 [KayKit](https://kaylousberg.itch.io/) CC0 美术资源 · Three.js 制作

## 🎮 开始游戏

**本地(推荐,支持局域网联机):**
双击 `启动.command`(或 `node server.mjs` 后打开 http://localhost:8996)
> 若提示无法打开/没有权限:在终端执行 `chmod +x 启动.command` 后再双击;或右键 → 打开。
> 找不到 node 时会自动降级为 python3 静态服务(游戏完整可玩,联机走同设备多标签通道)。

**在线版:** https://frankilito.github.io/hogwarts-legacy-gaiden/
> 首次进入需下载约 25MB 核心资源(其余角色后台静默补齐),弱网会自动重试;
> 游戏带「魔法缓存」(Service Worker)——下过的资源不再重复下载,第二次进入基本秒开。
> 若网络对 github.io 不稳,多刷新一两次即可(每次刷新都会保留已下载的部分)。

**免服务器直连入口:** 双击项目里的 `网页CDN入口.html` —— 整个游戏直接从 jsDelivr CDN 流式加载,
不需要本地服务器、不经过 github.io(对国内网络最稳),这个小文件也可以直接发给朋友。

## 📖 你将体验

- **创建你的巫师**:四大学院分院、魔法专长、性格、体型/发型/肤色定制
- **学院生活**:六阶段昼夜与按日推进的课表 —— 魔咒课(符文描绘)、魔药课(投料·搅拌·火候)、草药课(曼德拉草 QTE)、天文课(星图连线)、黑魔法防御术,每周末学期考试(笔试+实践)
- **角色关系**:14 位性格各异的 NPC(教授/同学/幽灵/小精灵),好感五阶,送礼、闲聊、同伴任务;挚友可随行冒险,合力发动**双人合击咒**
- **主线《沉睡的机关》**:两百年前创校工匠留下的地下机关苏醒了——集齐学院之钥,潜入地下密室,面对古代守卫,做出你的抉择(双结局)
- **实时魔法战斗**:魔弹/昏迷咒/铁甲咒(完美格挡反弹!)/火焰/冰冻/雷电链/漂浮/变形术/时空门;锁定、翻滚、决斗社晋级赛
- **地下迷宫与机关**:压力板浮石、双火盆、古代传送门、宝箱金币、骷髅苏醒
- **禁林探索**:夜采月光花、捕萤火、幽灵怀表委托
- **宿舍装饰**:17 种家具自由摆放,朋友联机来参观
- **技能树**:元素/守护/奥秘三系 13 节点
- **联机**:参观宿舍 / 庭院决斗圈 PvP / 密室禁林协同作战(本地服务器或同设备多标签)

## ⌨️ 操作

| 按键 | 功能 |
|---|---|
| WASD | 移动(Shift 潜行) |
| 鼠标 | 视角(点击画面锁定指针) / 滚轮缩放 |
| 左键 | 魔弹 · 右键/C | 护盾(掐点完美格挡) |
| 1-4 | 咒语槽 · Q 锁定 · F 同伴合击 |
| 空格 | 翻滚闪避 / 对话继续 |
| E | 交互 · Tab 主面板 · R/Q 装饰模式旋转/移除 |

移动端:虚拟摇杆 + 触屏按钮。

## 🛠 技术

- Three.js r170 · 原生 ES Modules,零构建
- 手写 WebSocket 服务器(零依赖)+ BroadcastChannel 双通道联机
- 程序化:石墙/彩窗/书墙/羊皮纸纹理、会动的画像(目光追随+眨眼+说话)、魔法天花板、体积光、粒子系统、WebAudio 作曲与全部音效
- KayKit CC0 模型:9 角色(95 动画/只:施法×5、坐读躺、受击、格挡、苏醒…)+ 100+ 地牢建筑件
- 测试:`node tools/shoot.mjs shot out.png <zone>` 无头截图 / 自动化验证
